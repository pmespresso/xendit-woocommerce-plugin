A Wordpress Plugin that Simplifies Process Of Editing Product's Prices in Woocommerce

== Description ==

**Woocommerce Products Price Bulk Edit** lets you to edit Woocommerce Product Prices via 
a simple Admin Page inside Woocommerce Admin Menu.

= Update Products in one Place =
easily find and update Product's Prices in one page without having to scroll or navigate contents specially 
when you have thousands of products.

= You Have Lots of Products? so This Plugin is for you =
this plugin optimized and tested with thousans of products! you can easily navigate through them and
change their prices or titles, updates or delete them.


== Installation ==

for installing every Wordpress plugin you should do these following steps:

1. Decompress the package you downloaded and upload it to your /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use The Plugin!
